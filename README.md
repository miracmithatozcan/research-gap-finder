# ISEF Research Hub 🌌

**DeepMind Experimental Research Hub | Büyüleyici ISEF Projeleri Keşif Portalı**

Bu proje, lise ve üniversite öğrencilerinin uluslararası yarışmalara (özellikle ISEF hedefleri doğrultusunda) hazırlanırken, derin teknoloji (Deep Tech) ve ileri bilim (Advanced Science) literatüründeki güncel ve çözülmemiş araştırma açıklarını keşfetmelerini sağlayan modern bir web arayüzüdür.

## 🚀 Özellikler

- **Gelişmiş Filtreleme:** Fizik, Kimya, Biyoloji ve Teknoloji & Tasarım gibi alanlara doğrudan erişim.
- **Dinamik Algoritma Arama:** Konu, domain veya spesifik algoritma üzerine anında canlı arama.
- **Doğrulanmış Literatür:** IEEE, Nature, Joule gibi dergilerden alınmış güncel ve yüksek etki faktörlü araştırma boşlukları.
- **Favorilere Ekleme:** Beğenilen projeleri `localStorage` altyapısı ile tarayıcı hafızasına alıp dilediğiniz zaman erişme imkânı.
- **Rastgele Fikir Üreteci (RefreshCw):** Özel algoritma yardımıyla veritabanında yer alan birbirinden yaratıcı fikirlere "Zardan" atar gibi ulaşım.
- **Büyüleyici Tasarım:** Glassmorphism arayüz (Vanilla CSS), kaydırılabilir modaller, süzülen atom animasyonları ve kademeli (staggered) render teknolojisi.

## 🛠️ Teknolojiler

- **React.js** + **Vite** (Geliştirme Ortamı)
- **Vanilla CSS** (Balyoz gücünde saf CSS keyframe animasyonları)
- **Lucide-React** (Vektörel ikon kütüphanesi)
- **Local Storage API** (Veri kalıcılığı)

## 📦 Kurulum ve Çalıştırma

Projeyi kendi bilgisayarınızda çalıştırmak için:

1. Bu projeyi indirin veya kopyalayın (Clone):
```bash
git clone <sizin-repo-linkiniz>
```

2. Proje dizinine girin:
```bash
cd isef-portal
```

3. Gerekli kütüphaneleri kurun:
```bash
npm install
```

4. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

5. Tarayıcınızda [http://localhost:5173/](http://localhost:5173/) adresine gidin.
